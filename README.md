## Description

[Nestjs](https://github.com/nestjs/nest) framework based jwt authentication.

## Swagger URL

[http://localhost:3000/api](http://localhost:3000/api)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
